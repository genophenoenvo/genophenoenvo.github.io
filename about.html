Welcome to the GenoPhenoEnvo Project

Project Name

Genotypes and Environments result in the Phenotypes of organisms

Our project is so named after Genomics - Phenotyping - and Environment data that we're using to create predictive models using interpretable machine learning.

Who We Are

We are a Collaborative Research Project spread out across four different institutions of higher learning.
